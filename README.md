# U.S. Electric Vehicle Market Share Analysis
This project analyzes electric vehicle adoption trends across U.S. states using SQL and Tableau. The analysis explores EV market share, alternative fuel adoption, geographic trends, and state-level comparisons to identify regions leading the transition toward electric mobility.

## Tools & Technologies
- SQL
- Tableau
- Excel

## Key Objectives
- Analyze EV adoption across U.S. states
- Calculate EV market share by state
- Compare EV adoption among major states
- Evaluate alternative fuel usage
- Build an interactive Tableau dashboard

## Key Insights
- California and northeastern states lead EV adoption.
- Southern states continue to lag in EV penetration.
- CNG shows the strongest alternative fuel adoption.
- Hydrogen vehicle adoption remains highly niche.
